# 📚 EstudoLog Pro - Santander Open Academy

Projeto desenvolvido para consolidar os conhecimentos adquiridos no curso de **Python** do **Santander Open Academy** (concluído em 16/04/2026).

## 🚀 Funcionalidades
- Registro de matérias estudadas com carimbo de data/hora automático.
- Geração de relatórios de matérias únicas (evitando duplicatas).
- Sistema de feedback motivacional aleatório.
- Exclusão seletiva ou total do histórico.

## 🛠️ Conceitos Aplicados
- **Modularização Profissional**: Divisão do sistema em pacotes e módulos para melhor manutenção.
- **Tratamento de Exceções**: Uso de `try/except` para prevenir erros de entrada do usuário.
- **Persistência de Dados**: Leitura e escrita em arquivos `.txt` usando o gerenciador de contexto `with`.
- **Estruturas de Dados**: Uso de `Dictionaries` para organização e `Sets` para filtragem de dados únicos.

## 📂 Estrutura do Projeto
- `main.py`: Ponto de entrada do sistema.
- `core/`: Pacote contendo a lógica de processamento, persistência e deleção.
